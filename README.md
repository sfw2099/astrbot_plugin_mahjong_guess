# 立直麻将猜胡牌

![Version](https://img.shields.io/badge/version-v0.2.2-blue)
![Python](https://img.shields.io/badge/python-3.10+-green)

AstrBot 插件：**Wordle 风格的立直麻将猜胡牌游戏**。随机生成一组带役种的胡牌牌型，玩家输入猜测，系统返回颜色标记反馈。

## 游戏规则

- 系统随机生成一组符合日本立直麻将规则的 14 张胡牌手（至少一个役种）
- 玩家用特定格式输入猜测：`w112233 s445566 p77`
- 猜中后返回颜色反馈图片：

| 颜色 | 含义 |
|------|------|
| 绿色 | 正确牌型，位置正确 |
| 黄色 | 正确牌型，位置错误 |
| 灰色 | 不在胡牌手中 |

- 最多 10 次机会（可配）

## 指令

| 指令 | 说明 |
|------|------|
| `/猜胡牌` | 开始新游戏 |
| `/随机胡牌 [役种]` | 随机练习，可选指定役种（如 `/随机胡牌 国士无双`） |
| `w... s... p... z...` | 发牌型猜测 |
| `结束猜胡牌` | 结束当前游戏 |

## 手牌格式

手牌由 4 个花色组组成，每组格式为 `花色 + 数字序列`：

| 花色 | 字母 | 范围 | 示例 |
|------|------|------|------|
| 万 | w | 1-9 | `w112233` = 一万×2, 二万×2, 三万×2 |
| 筒 | p | 1-9 | `p77` = 七筒×2 |
| 索 | s | 1-9 | `s445566` = 四索×2, 五索×2, 六索×2 |
| 字 | z | 1-7 或 東南西北白発中 | `z東東東` / `z111` = 東×3 |

手牌必须有 14 张。**最后一张是河底牌**（获胜牌），不参与排序。

## 支持的役种

| 役种 | 番 | 说明 |
|------|----|------|
| 断幺九 | 1番 | 全 2-8，无幺九牌和字牌 |
| 役牌 | 1番 | 含三元牌/场风/自风刻子 |
| 平和 | 1番 | 4顺子 + 非役牌雀头，两面听 |
| 一盃口 | 1番 | 两组完全相同的顺子 |
| 对对和 | 2番 | 4刻子 + 1雀头 |
| 七对子 | 2番 | 7组不同的对子 |
| 三色同顺 | 2番 | 万筒索三色相同顺子 |
| 一气通贯 | 2番 | 同花色的 1~9 一条龙 |
| 三色同刻 | 2番 | 万筒索三色相同刻子 |
| 混一色 | 3番 | 一种数牌 + 字牌 |
| 小四喜 | 役满 | 3种风牌刻子 + 1种风牌雀头 |
| 字一色 | 役满 | 全部由字牌组成 |
| 国士无双 | 役满 | 13种幺九牌各1张 + 任意1张重複 |

## 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| max_attempts | 10 | 每局最大猜测次数 |
| show_yaku_hint | true | 开始时显示役种提示 |
